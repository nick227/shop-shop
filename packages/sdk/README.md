# SDK Package

This package provides a TypeScript SDK for the shop-shop API, with clear separation between generated and source files.

## 🏗️ Structure

```
packages/sdk/
├── generated/                    # 🔴 GENERATED FILES (DO NOT EDIT)
│   ├── sdk/                     # Raw OpenAPI generated files
│   ├── frontend/                # Frontend-specific generated files
│   └── scripts/                 # Generated configuration files
├── src/                         # 🟢 SOURCE FILES (EDITABLE)
│   ├── index.ts                 # Main entry point
│   └── realtime/                # Realtime functionality
├── scripts/                     # 🟢 GENERATION SCRIPTS (EDITABLE)
└── dist/                        # Compiled output
```

## 🚀 Quick Start

### Development
```bash
# Generate all files
pnpm gen:all

# Generate specific components
pnpm gen:wrapper    # API wrappers
pnpm gen:hooks      # React hooks
pnpm gen:types      # Type definitions
```

### Migration
```bash
# Migrate to new structure
pnpm migrate
```

## 📁 File Categories

### 🔴 Generated Files (DO NOT EDIT)
- All files in `generated/` directory
- Files with `AUTO-GENERATED` headers
- Files generated by OpenAPI tools
- Files generated by custom scripts

### 🟢 Source Files (EDITABLE)
- Files in `src/` directory
- Files in `scripts/` directory
- Configuration files
- Documentation files

## 🔄 Generation Workflow

1. **OpenAPI Generation**: `pnpm gen` → `generated/sdk/`
2. **Frontend Types**: `pnpm gen:frontend-types` → `generated/frontend/types/`
3. **API Wrappers**: `pnpm gen:wrapper` → `generated/frontend/api/`
4. **React Hooks**: `pnpm gen:hooks` → `generated/frontend/hooks/`
5. **Build**: `pnpm build` → `dist/`

## 📦 Usage

### From Apps
```typescript
// Generated API wrappers
import { apiWrapper } from '@packages/sdk/generated/frontend/api/apiWrapper'

// Generated types
import type { StoreResponse } from '@packages/sdk/generated/frontend/types/backend-types'

// Generated hooks
import { useStores } from '@packages/sdk/generated/frontend/hooks/generated'

// Source utilities
import { createApiClient } from '@packages/sdk'
```

### From SDK Package
```typescript
// Re-export generated files
export * from './generated/frontend/api/apiWrapper'
export * from './generated/frontend/types/backend-types'
export * from './generated/frontend/hooks/generated'

// Source files
export * from './src/realtime'
```

## 🛠️ Development

### Adding New Generators
1. Create script in `scripts/`
2. Add to `package.json` scripts
3. Update `GENERATION_STRUCTURE.md`
4. Test with `pnpm gen:all`

### Modifying Generated Files
❌ **Never edit files in `generated/` directly**
✅ **Modify the generation scripts instead**

## 📋 Available Scripts

- `pnpm gen` - Generate raw SDK from OpenAPI
- `pnpm gen:wrapper` - Generate API wrappers
- `pnpm gen:hooks` - Generate React hooks
- `pnpm gen:types` - Generate type definitions
- `pnpm gen:all` - Generate all files
- `pnpm migrate` - Migrate to new structure
- `pnpm build` - Build the package
- `pnpm validate` - Validate generated files

## 🔧 Configuration

- `openapitools.json` - OpenAPI generator configuration
- `tsconfig.json` - TypeScript configuration
- `scripts/paths.ts` - Path constants for generators
- `scripts/resource-config.ts` - Resource configuration

## ⚠️ Important Notes

1. **Never edit generated files** - They will be overwritten
2. **Use migration script** - Don't manually move files
3. **Update imports** - After migration, update import paths
4. **Test thoroughly** - Verify all functionality after changes
5. **Commit structure** - Commit the new structure, not generated content
